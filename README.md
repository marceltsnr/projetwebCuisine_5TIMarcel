## 📝 Description du projet

Ce projet est une application web de **partage de recettes de cuisine**. Il permet aux utilisateurs de s'inscrire, de publier leurs propres créations culinaires et de consulter celles de la communauté. 

L'objectif pédagogique est de maîtriser le **CRUD** (Create, Read, Update, Delete) avec PHP et une base de données MySQL.

### ✨ Fonctionnalités clés
- 🔐 **Authentification** : Inscription et connexion sécurisée
- 📖 **Catalogue** : Consultation de toutes les recettes avec affichage par catégorie.
- ✍️ **Gestion (CRUD)** : Ajouter, modifier et supprimer ses recettes.
- 📱 **Responsive** : Interface adaptée aux mobiles et tablettes.
